# Data-Engineering---The-Sentinel-Server-Log-Analyzer
A Python-based log analysis pipeline that parses raw, unstructured server logs to detect security threats and diagnose performance bottlenecks. This project builds a complete ETL (Extract, Transform, Load) pipeline from scratch — using only Python — to turn raw, messy stream of data into actionable insights.

# Concepts & Skills Demonstrated

- String cleaning and manipulation
- Data structures: lists of dictionaries as records
- Type casting and data sanitization (string → int, unit conversion)
- Writing reusable functions with optional parameters
- Filtering and aggregation logic (counting, averaging)
- Dictionary-based counting patterns
- Formatted output with f-strings
- List comprehensions (alternative implementation included)
